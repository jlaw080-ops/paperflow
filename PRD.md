# PRD: PaperFlow (마크다운 문서 웹 퍼블리싱 앱)

> 작업용 임시 이름이 PaperFlow입니다. 마음에 드는 이름으로 바꿔도 됩니다.

## 목표
Obsidian 등에서 작성한 마크다운 문서를 웹에서 보기 좋게 정리·편집하고, 링크 하나로 공유하며, 인쇄·PDF 저장 시 각 페이지가 A4 크기로 깔끔하게 나뉘는 **개인용 문서 퍼블리싱 도구**를 만든다.

## 사용자
- **편집자(주 사용자)**: 본인 1명. 건축설비·신재생에너지(ZEB, Passive house, 건물에너지 VE) 분야 기술 문서를 작성·정리한다.
- **열람자**: 공유 링크를 받은 누구나(동료, 발주처, 협력사 등). 로그인 없이 본다.

## 핵심 기능 (MVP)
1. **문서 가져오기** — `.md` 파일을 끌어다 놓아(드래그앤드롭) 앱으로 가져온다.
2. **폴더 트리** — 좌측 사이드바에서 업무/프로젝트별로 폴더·문서를 만들고, 이름 변경·이동하며, 트리로 탐색한다.
3. **앱 내 마크다운 편집** — 가져온 문서나 새 문서를 앱에서 직접 작성·수정·저장한다.
4. **공개 보기 페이지** — 공유 링크(URL)로 누구나 열람한다. 화면에서는 일반 웹페이지처럼 세로로 쭉 이어진다.
5. **인쇄/PDF A4 분할** — 인쇄하거나 PDF로 저장할 때 각 페이지가 A4 크기로 구분된다.
6. **소유자 로그인** — 편집은 본인만 가능(간단한 이메일 로그인). 열람은 누구나 가능.

## MVP 제외 사항
> 이걸 명시해 두지 않으면 AI가 기능을 끝없이 늘린다. "안 만든다"를 분명히 한다.

- **Obsidian 자동 양방향 동기화** — 가져오기는 1회성 import만. 이후 원본은 앱이다.
- **Obsidian 전용 문법** — `[[위키링크]]`, `![[임베드]]`, 콜아웃(callout) 등은 지원하지 않는다. **표준 마크다운(GFM, 표 포함)만** 지원한다.
- **여러 사용자 동시 편집·협업, 댓글, 변경 이력(버전 관리)**
- **문서별 개별 접근 권한** — 열람은 전부 공개로 통일한다.
- **전체 문서 검색, 태그 시스템**
- **수식(LaTeX), 다이어그램(Mermaid) 렌더링** — 후순위로 미룬다.
- **이미지 업로드/호스팅** — MVP에서는 외부 URL 이미지만 표시. (직접 업로드는 다음 단계)

## 디자인 방향
- 기술 보고서에 어울리는, 차분하고 가독성 높은 **"종이 같은" 디자인**. 인쇄 친화적.
- 거의 흰 배경 + 진한 회색 본문, 절제된 강조색. (상세: `UI_GUIDE.md`)
- AI 슬롭 패턴(보라색 테마, 글래스모피즘, 네온 글로우 등) 금지. (상세: `UI_GUIDE.md`)

## 성공 기준 (이게 되면 MVP 완성)
- [ ] `.md` 파일을 드래그앤드롭으로 가져오면 트리에 문서가 생긴다.
- [ ] 폴더를 만들고 문서를 그 안으로 옮길 수 있다.
- [ ] 앱에서 문서를 수정·저장하면 내용이 유지된다.
- [ ] 공유 링크를 로그아웃 상태(또는 다른 브라우저)에서 열면 문서가 보인다.
- [ ] 그 페이지를 인쇄 미리보기로 열면 A4 단위로 페이지가 나뉜다.
- [ ] 로그인하지 않으면 편집 화면에 접근할 수 없다.
